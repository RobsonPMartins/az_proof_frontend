# AZ Suite Proof

API para acompanhamento de pedidos e valores recebidos.

## Tecnologias
- Node.js
- Express
- MongoDB
- Mongoose

## Instalação
```sh
https://github.com/RobsonPMartins/az_proof_frontend.git
```

```
cd order-tracking-api
```
## Intale as dependências
```
npm install
```
