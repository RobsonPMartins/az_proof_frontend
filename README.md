# AZ Suite Proof

API para acompanhamento de pedidos e valores recebidos.

## Tecnologias
- Node.js
- Express
- MongoDB
- Mongoose

## Instalação
```sh
git clone https://github.com/seu-usuario/az_proof_frontend
.git
```
## Entrar na pasta
```
cd order-tracking-api
```
## Intale as dependências
```
npm install
```
